O Saúde Fácil é um sistema de gestão hospitalar desenvolvido em arquitetura full-stack, utilizando TypeScript, ExpressJS, Prisma e PostgreSQL com Docker para o backend. A API segue o padrão REST e conta com validação de dados via Zod, autenticação JWT e controle de acesso por níveis (administrador e recepcionista). A documentação e os testes dos endpoints são realizados pelo Swagger, permitindo visualizar e verificar dados de forma prática. Como uma API funciona por meio de regras e protocolos de comunicação entre sistemas, ela não possui interface visual própria, mas é essencial para integrações, operações CRUD e consulta de informações de pacientes, médicos, clínicas e consultas. No frontend, o sistema utiliza React, TypeScript, Vite e Tailwind CSS, oferecendo uma interface moderna, responsiva e integrada com a API para gerenciar todo o fluxo hospitalar.
